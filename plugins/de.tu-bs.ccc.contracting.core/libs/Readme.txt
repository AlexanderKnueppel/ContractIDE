# Download antlr-4.7.1-complete.jar

http://www.antlr.org/download/antlr-4.7.1-complete.jar

# Install z3

* Download apropriate release: https://github.com/Z3Prover/z3/releases
* Extract folder into this libs folder
* Rename folder to z3 (i.e., such that z3/include and z3/bin exist)
* On windows, you propably need to add the path to the bin folder to your global environment (z3/bin/).

# Add to build path

In case of errors, add both jars manually to the build path
